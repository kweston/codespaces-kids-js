# Kids JavaScript Projects for GitHub Codespaces

This repo includes 5 beginner-friendly JavaScript projects that work well for two learners at different levels.

## Projects

1. **guessing-game** - Number guessing game in the terminal
2. **rps** - Rock, paper, scissors in the terminal
3. **todo** - Command-line to-do list with file storage
4. **web** - A simple browser-based counter app
5. **chatbot** - A rule-based mini chatbot in the terminal

## Quick start in Codespaces

1. Create a new GitHub repo and upload these files.
2. Open the repo in **GitHub Codespaces**.
3. Wait for setup to finish.
4. Open the terminal and run one of these:

```bash
npm run guess
npm run rps
npm run todo
npm run chatbot
npm run web
```

For the web project, Codespaces should automatically forward port **3000** and open the page.

## Suggested teamwork

- **11-year-old:** type the code, test changes, suggest fun ideas
- **16-year-old:** help plan the logic, debug issues, add stretch features
- Switch roles every 10 to 15 minutes.

## Challenge ideas

### 1) Guessing Game
- Change the random number range
- Add a score
- Add difficulty levels

### 2) Rock Paper Scissors
- First to 3 wins
- Keep score across rounds
- Add `lizard` and `spock`

### 3) To-Do List
- Mark tasks as complete
- Add due dates
- Sort by status

### 4) Web Counter
- Add a decrement button
- Add a reset button
- Save the count in local storage

### 5) Chatbot
- Add more commands
- Respond differently based on keywords
- Add a help menu

Have fun and keep the rule simple: **make it work first, then make it cooler.**
